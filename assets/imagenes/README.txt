INSTRUCCIONES PARA LA IMAGEN DE FONDO
=======================================

Para que el login se vea completo, necesitas agregar una imagen de fondo.

OPCIÓN 1: Usar tu propia imagen
--------------------------------
1. Coloca tu imagen en esta carpeta
2. Renómbrala a: 2.png
3. La imagen debe ser de alta calidad (recomendado: 1920x1080 o superior)
4. Formatos aceptados: PNG, JPG, JPEG

OPCIÓN 2: Usar una imagen de prueba
------------------------------------
Puedes descargar una imagen de tecnología/oficina desde:
- Unsplash: https://unsplash.com/s/photos/office-technology
- Pexels: https://www.pexels.com/search/printer/
- Pixabay: https://pixabay.com/images/search/office/

OPCIÓN 3: Sin imagen
---------------------
Si no agregas una imagen, el login funcionará perfectamente con el gradiente
de fondo azul que ya está configurado.

RUTA COMPLETA:
c:\xampp\htdocs\impresoras\assets\imagenes\2.png

Una vez que coloques la imagen, recarga el navegador (Ctrl+F5) para ver los cambios.
